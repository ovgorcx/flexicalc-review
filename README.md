## 为什么普通计算器解决不了这个问题

想象一个场景：你在算这个月的房贷。

利率从 3.9% 变成了 3.7%，月供从 7300 变成 7150，三十年总利息差了将近七万。你想把两个方案放在一起对比——普通计算器做不到，你只能重算，或者在备忘录里手动粘贴数字。

灵活计算器的解法是：建立方案 A，复制一份做方案 B，改掉利率，两套结果并排放在同一页里。利率再变，直接改那个数字，其他都跟着动。

这不是什么魔法，但是很有用。用小众软件论坛用户的话说，它就是"升级版的计算器 + 便签"。

## 核心功能逐一拆解

### 实时关联计算

这是整个 app 最重要的特性。任何一个数字被修改后，所有依赖这个数字的计算结果会立即自动更新。不用按任何按钮，不用重新输入，直接改完就好。

用于贷款测算、预算规划、成本利润分析这类需要反复调参的场景，省的时间非常明显。

### 自由拖动排版

数字、运算符、整行表达式都可以拖动到任意位置。想把"月供"和"总利息"放在一起看，拖过去就行。

这个功能在处理多步骤计算时特别方便，因为你可以按照自己的逻辑去组织计算顺序，而不是被迫从上到下线性排列。

### 数字标签

为每个数字添加文字说明或 Emoji 标签，比如"本金""税率""折扣后价格"。几天后打开这个计算页，每个数字代表什么一目了然。

这个功能解决了一个真实痛点：计算器里的历史记录只有数字，完全不知道那串数字是算什么的。

### 模板库

内置贷款、投资收益、折扣价格、水电费等常用模板，打开直接用。更重要的是，你可以把自己常用的计算结构保存成模板，下次不用重头搭建公式框架。

### 云端同步与多设备协作

所有计算历史、表达式、标签都可以云端保存，多设备之间自动同步。手机上起草的预算，平板上继续改，Mac 上导出分享。
